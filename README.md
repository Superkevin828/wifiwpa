# WiFi Auditor v2.0

Cybersecurity WiFi Auditor Suite with professional GUI interface.

## Features
- Network Scanning: Discover nearby WiFi networks
- WPA Handshake Capture: Capture WPA/WPA2 handshakes
- Password Cracking: Built-in password cracking with wordlist support
- Wordlist Generator: Create custom wordlists
- Professional GUI: Cyberpunk-themed command center interface

## Requirements
- Linux: Python 3.8+, aircrack-ng, gnome-terminal, python3-tk
- Windows: Python 3.8+, aircrack-ng (via WSL or native)
